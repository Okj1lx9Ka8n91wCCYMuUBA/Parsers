
# Парсер Корпорация МСП corpmsp.ru

## Запуск

```bash
pip install -r ./../requirements.txt
```

Параметр `USE_KAFKA` позволяет запустить файл `main.py` как скрипт без использования `kafka`.

```bash
python ./corpmsp/main.py
```
