Назначение
Предоставление льготы по налогу на прибыль - 2 % для организаций, осуществляющих производство электроэнергии тепловыми электростанциями
Предоставление льготы по налогу на прибыль - 3 %
для управляющих компаний индустриальных (промышленных) парков
Предоставление льготы по налогу на прибыль - 4 % для социально ориентированными некоммерческими организациями
Предоставление льготы по налогу на прибыль - 7 % для участников региональных инвестиционных проектов ( с января 2020 г.)
Предоставление льготы по налогу на прибыль - 8 % для участников специальных инвестиционных контрактов
