Назначение
Предоставление поручительств по обязательствам субъектов малого и среднего предпринимательства Новосибирской области, реализующих проекты развития и расширения бизнеса, вытекающих из заключаемых ими кредитных договоров, договоров займа, лизинга (финансовой аренды), о предоставлении банковской гарантии, и иных договоров, предусмотренных действующим законодательством Российской Федерации.

Максимальный размер ответственности Фонда по отношению к обеспечиваемому обязательству не может превышать 50% от суммы основного долга по финансовому обязательству при предоставлении поручительства Фонда Субъектам по продуктам Фонда:
- «Поручительство до 5 млн. рублей по Кредитным договорам»;
- «Поручительство до 25 млн. рублей по Кредитным договорам»;
- «Поручительство по кредитным договорам до 50% от суммы кредита»;
- «Поручительство по договорам займа с МФО»;
- «Поручительство по Действующим Кредитным договорам»;
- «Поручительство по Рефинансируемым/Замещаемым Кредитным договорам».
Максимальный размер ответственности Фонда по отношению к обеспечиваемому обязательству не может превышать 70% от суммы основного долга по финансовому обязательству при предоставлении поручительства Фонда Субъектам по продуктам Фонда:
- «Поручительство по Кредитным договорам Основная программа»*;
- «Поручительство на инвестиционные цели на период не более ½ срока кредита»*;
- «Поручительство по договорам Банковской гарантии»*;
- «Поручительство по договорам Банковской гарантии в рамках №44-ФЗ и №223-ФЗ»;
- «Поручительство по договорам займа с ФРП»;
- «Поручительство по Кредитным договорам по продукту Согарантия»;
- «Поручительство по Кредитным договорам по продукту Согарантия до 75%».
*до 60% субъектам МСП основным видом деятельности, которых является оптовая и розничная торговля; операции с недвижимым имуществом, аренда; организация комплексного туристического обслуживания.
