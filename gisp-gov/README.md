
# Парсер Фонд развития промышленности ([frprf.ru](frprf.ru))

## Запуск

```bash
pip install -r ./../requirements.txt
```

Параметр `USE_KAFKA` позволяет запустить файл `main.py` как скрипт без использования `kafka`. В отличии от других парсеров, этот требует обязательное наличие поля номера страницы `page_number` в сообдении в топик `gisp_parse_tasks`.

```bash
python ./gisp-gov/main.py
```
